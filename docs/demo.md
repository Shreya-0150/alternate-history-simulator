# Demo Guide

## How to Run a Simulation

1. Open Telegram
2. Find @butterfly_effect_bot
3. Type /simulate followed by your question

## Example Simulations

### Example 1: Gandhi
```
/simulate What if Gandhi was never born?
```

### Example 2: Internet
```
/simulate What if the internet was never invented?
```

### Example 3: WW2
```
/simulate What if World War 2 never happened?
```

### Example 4: India
```
/simulate What if India was never colonized?
```

## What You Get Back
- Historical accuracy score
- Dramatic alternate history story
- Second and third order ripple effects
- Full alternate timeline

## Commands
- /simulate [question] — Run simulation
- /help — Show help
